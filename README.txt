== Description ==

TP Elementor Addons Kits Plugin that show profile card details on post or page with user's social links.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Elementor Addons Kits'
3. Activate Elementor Addons Kits from your Plugins page.

= From WordPress.org =

1. Download TP Elementor Addons Kits.
2. Upload the 'tp-elementor-addons-kits' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate TP Elementor Addons Kits from your Plugins page.
