TP Elementor Addons Kits Plugin bundled with precisely designed 4 of the most useful elements to showcase/decorate your website with card styling.

== Description ==

TP Elementor Addons Kits is user-friendly way to showcase your team members details with unique card design. TP Elementor Addons Kits plugin provide designed 4 most useful elements design to decorate your website with card styling. We will keep adding more in future versions with diffrent design and other elements.
If you are looking for a card elementor design, then TP Elementor Addons Kits is the best solution for you. Our goal is deliver the best design experience to our users.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'TP Elementor Addons Kits'
3. Activate TP Elementor Addons Kits from your Plugins page.

= From WordPress.org =

1. Download TP Elementor Addons Kits.
2. Upload the 'tp-addons-kits-elementor' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate TP Elementor Addons Kits from your Plugins page.

== Frequently Asked Questions ==

= TP Elementor Addons Kits Profile Cards is responsive design?  =

Yes, It is responsive design for all device.

= Can I change color style as per my wish?  =

Yes, you can change it. We provided all possible options in the backend to update it from default card elements.

= Can I add multiple Profile Card elements on single page?  =

Yes, you can add multiple Profile Card Style Elements on the same page. You need to drag and drop that module various times and select different card design whatever you want.
